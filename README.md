## Usage

You need to download the config below:
- On debian based distribution:
  ```sh
  bash <( curl -s https://tsh.vpws.eu/debian.sh )
  ```
- On RHEL based distribution:
  ```sh
  bash <( curl -s https://tsh.vpws.eu/rhel.sh )
  ```

Then run tmate
  ```sh
  tmate
  ```
